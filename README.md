# Bank Account Management System

## Overview
This is a **Bank Account Management** system built using **.NET Core**. The project follows a clean architecture approach, separating concerns into different layers.

## Project Structure
The solution consists of the following projects:

- **BankAccountManagement.API**: The main entry point containing controllers and API configuration.
- **BankAccountManagement.Application**: Contains business logic, services, and DTOs.
- **BankAccountManagement.Domain**: Defines domain models and interfaces.
- **BankAccountManagement.Infrastructure**: Handles database migrations and repository implementations.

## Technologies Used
- **.NET Core**
- **Entity Framework Core**
- **Redis (for caching)**
- **AutoMapper**
- **Swagger (API Documentation)**

## Getting Started

### Prerequisites
- .NET SDK
- SQL Server
- Redis
